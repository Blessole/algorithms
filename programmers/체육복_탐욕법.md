# 체육복

## 문제 설명

- **출처** : 프로그래머스
- **난이도** : LV1
- **링크** : https://school.programmers.co.kr/learn/courses/30/lessons/42862

[문제 요약]
학생들이 체육복을 도난당했는데, 여벌 있는 학생이 빌려줄 수 있다. 단, 바로 앞번호나 뒷번호 학생에게만 빌려줄 수 있다. 체육수업을 들을 수 있는 최대 학생 수를 구한다.

## 내 풀이

### 접근 방법

1. 학생 수 + 1 크기의 배열을 만들어 모두 1(체육복 1개)로 초기화한다
2. lost 학생은 -1, reserve 학생은 +1 해서 체육복 개수를 표현한다
3. 앞에서부터 순회하며 여벌(2개) 가진 학생이 앞/뒤 학생 중 0개인 학생에게 빌려준다
4. 체육복 1개 이상 가진 학생 수를 반환한다

### 내 코드

```javascript
function solution(n, lost, reserve) {
    let studentList = Array(n+1).fill(1);
    lost.forEach(l => studentList[l]--);
    reserve.forEach(r => studentList[r]++);
    
    for (let i=1; i<=n; i++) {
        if (studentList[i] > 1) {
            if (studentList[i-1] === 0) {
                studentList[i-1]++;
                studentList[i]--;
            } else if (studentList[i+1] === 0) {
                studentList[i+1]++;
                studentList[i]--;
            }
        }
    }
    return studentList.slice(1).filter(s => s >= 1).length;
}

## 정리

### Array 활용
- Array(n).fill(값) : 특정 값으로 초기화된 배열 생성
- 인덱스=학생번호 로 맞추려면 Array(n+1)로 만들어 0번 인덱스는 사용하지 않는다.


### 회고
처음에 Map이나 객체 배열로 접근하려고 했는데, 단순 숫자 배열이 훨신 간단했다.
- 상태를 숫자로 표현하면 (0:없음, 1:있음, 2:여벌) 연산이 쉬워진다.
- n+1크기로 배열을 만들면 인덱스 계산 없이 학생 번호 그대로 접근이 가능하다!!
- 빌려주는 로직에서 실제로 빌려줬을 때만, studentList[i]-- 를 실행해야 버그가 줄어든다.
